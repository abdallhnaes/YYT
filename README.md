# تطبيق إدارة بسطة مازوت

تطبيق عربي يعمل بدون إنترنت داخل WebView، ومخصص لإدارة بسطة مازوت صغيرة.

## الميزات

- تهيئة أولية للكاش والمخزون ومتوسط كلفة اللتر.
- إضافة مبيعات مازوت نقدية أو دين.
- إضافة مشتريات من الموردين وحساب الدين عليهم.
- إدارة الأشخاص: زبائن وموردين.
- دفعات على الحساب: تحصيل من زبون أو دفع لمورد.
- إدارة الكاش: إضافة، سحب، مصاريف.
- تسوية المخزون: نقص، زيادة، تثبيت كمية.
- تقارير حسب الفترة مع طباعة.
- نسخ احتياطي واستعادة JSON.
- تصدير CSV.
- رمز PIN اختياري.
- يعمل بدون قاعدة بيانات خارجية، ويخزن البيانات على الجهاز عبر localStorage.

## طريقة رفعه على GitHub وبناء APK

1. أنشئ مستودع جديد على GitHub.
2. ارفع كل ملفات هذا المجلد إلى المستودع.
3. افتح تبويب Actions.
4. شغّل workflow باسم `Build Diesel Stand APK`.
5. بعد انتهاء البناء ستجد ملف APK داخل Artifacts باسم `DieselStand-debug-apk`.

## تعديل الاسم أو الألوان

يمكن تعديل الاسم من داخل التطبيق من صفحة الإعدادات، أو من ملف:

`app/src/main/assets/index.html`

## ملاحظة

هذا التطبيق نسخة أولية كاملة وقابلة للتوسيع، ويمكن إضافة طباعة فواتير منفصلة أو مزامنة سحابية لاحقًا.
